### Cara Install

1. **Download APK** menggunakan tombol atau link di atas
   - Jika download stuck di 99%, lihat [Troubleshooting Download](#-troubleshooting-download) di bawah
2. Buka **Settings** → **Security & Privacy** di perangkat Android
3. Enable **"Install from Unknown Sources"** atau **"Allow from this source"**
4. Buka file **app-release.apk** dari folder Downloads
5. Tap **Install** dan tunggu hingga selesai
6. Tap **Open** untuk membuka aplikasi

## 🛠️ Troubleshooting

### 🔧 Troubleshooting Download

#### Download Stuck di 99% atau Tidak Selesai?

**Solusi Cepat:**
1. **Clear browser cache** dan download ulang
2. **Gunakan browser lain** (Chrome, Firefox, Samsung Internet)
3. **Download via WiFi** instead of mobile data
4. **Restart browser** kemudian coba lagi
5. **Tutup tab/app lain** yang menggunakan bandwidth

**Metode Alternative - Download Manager (Recommended):**

Jika download sering stuck, gunakan aplikasi download manager:

1. Install **[ADM (Advanced Download Manager)](https://play.google.com/store/apps/details?id=com.dv.adm)** dari Play Store (GRATIS)
2. Copy link download:
```
   https://github.com/Bahana-Cipta-Technology/bss-opr-apk/releases/download/v1.0.0%2B27/app-release.apk
```
3. Buka ADM
4. Paste link di ADM
5. Download akan **otomatis resume** jika terputus
6. File akan tersimpan di folder Downloads

**Tips:**
- ✅ Download saat WiFi stabil
- ✅ Jangan minimize browser saat download
- ✅ Pastikan storage cukup (minimal 100MB free)
- ❌ Jangan pause/resume berkali-kali di browser biasa

---

### ❓ Tidak bisa download?
- Pastikan koneksi internet stabil
- Coba browser lain (Chrome, Firefox, dll)
- Disable VPN jika menggunakan
- Clear browser cache: Settings → Apps → Browser → Storage → Clear Cache
- Gunakan **ADM (Advanced Download Manager)** seperti instruksi di atas

### ❓ Tidak bisa install?
- Pastikan **"Unknown Sources"** sudah enabled
- Cek apakah storage device cukup (minimal 50MB free)
- Uninstall versi lama jika ada bentrok versi (lihat [Note untuk Update](#-note-untuk-update))
- Restart device dan coba lagi

### ❓ APK corrupt atau error saat install?
- Download ulang APK dari link di atas
- Pastikan download selesai 100% (tidak stuck/terputus)
- Gunakan **ADM** untuk download yang lebih stabil
- Verify ukuran file: APK harus sekitar 20-50MB
- Scan APK dengan antivirus jika perlu

### ❓ Aplikasi crash atau force close?
- Pastikan Android version minimal **5.0 (Lollipop)**
- Clear app cache: Settings → Apps → BSS OPR → Storage → Clear Cache
- Update ke versi terbaru
- Reinstall aplikasi (uninstall → install ulang)

### ❓ Error "App Not Installed" saat install?
Ini biasanya terjadi karena **signature mismatch** (bentrok dengan versi lama).

**Solusi:**
1. **Uninstall** aplikasi lama terlebih dahulu
2. **Download** APK versi baru
3. **Install** APK baru
4. **Login** kembali

**Catatan:** Data lokal akan hilang saat uninstall. Pastikan data sudah tersinkronisasi ke server.

## 🔄 Note untuk Update

### Update Pertama Kali dari Versi Lama

Jika Anda mengalami error **"App Not Installed"** saat update, berarti ada perubahan signing key.

**Cara Update:**
1. **Backup data** (pastikan data sudah sync ke server)
2. **Uninstall** aplikasi versi lama
3. **Download** APK baru dari link di atas
4. **Install** APK baru
5. **Login** kembali

**Catatan:**
- Update ini **hanya perlu dilakukan sekali**
- Update selanjutnya akan **otomatis** tanpa perlu uninstall
- Data lokal akan hilang, tapi data di server tetap aman
